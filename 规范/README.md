##前端规范##

**前言：**

前端主要由三大部分组成，即HTML+CSS+JS。原则其实也都很简单，就是简约、高效、复用、语意化。

规范是跟着W3C标准走，向<span style="color:#f00">HTML5</span>和<span style="color:#f00">CSS3</span>靠近。代码风格部分可参考BAT、谷歌等大型互联网公司看齐。当然选择认为合理的部分，而不是全盘接受。
